# P.I-Ofimatica Descripción

Este proyecto presenta el diseño de un sistema informático para la academia de inglés Learn to Learn, creado para mejorar la conectividad, la comunicación interna y el acceso compartido a recursos educativos y administrativos.

La infraestructura permite una red estable tanto cableada como inalámbrica, facilitando el trabajo diario del personal y el acceso de alumnos a internet y plataformas digitales.

Objetivos del Proyecto
Crear una red empresarial estable y segura.
Conectar todos los equipos de la academia.
Compartir recursos como impresoras y archivos.
Mejorar la productividad y la organización interna.
Preparar el sistema para futuras ampliaciones.
Infraestructura de Red
Router

Se encarga de gestionar el acceso a internet y proteger la red frente a amenazas externas.

Switch

Conecta las 7 estaciones de trabajo y la impresora mediante cable Ethernet, permitiendo una comunicación rápida y eficiente.

Access Point

Proporciona conexión Wi-Fi para alumnos y personal, permitiendo conectar dispositivos móviles como portátiles, tablets y smartphones.

Diseño de la Red

La red utiliza una topología en estrella, donde:

El router distribuye la conexión a internet.
El switch conecta todos los dispositivos cableados.
El access point ofrece cobertura inalámbrica en toda la academia.

Este diseño mejora la estabilidad y facilita futuras ampliaciones.

Sistema Operativo

Todas las estaciones utilizan Windows 11 Pro, ofreciendo:

Seguridad avanzada.
Compatibilidad con software educativo.
Facilidad de uso para profesores y administración.
Actualizaciones automáticas y soporte técnico.
Características Principales
Conectividad
Red rápida y estable.
Conexión simultánea de múltiples dispositivos.
Acceso inalámbrico y cableado sin interrupciones.
Recursos Compartidos
Impresora accesible desde cualquier PC.
Compartición de archivos entre departamentos.
Centralización de recursos para mejorar la colaboración.
Escalabilidad

El sistema está preparado para:

Añadir más ordenadores.
Mejorar la cobertura Wi-Fi.
Incrementar el ancho de banda según crezca la academia.
Comparación Antes y Después
Antes
Equipos aislados.
Internet lento e inestable.
Impresión limitada a un solo ordenador.
Dificultad para compartir archivos.
Después
Todos los equipos conectados en red.
Acceso estable a internet.
Impresora compartida.
Mayor productividad y comunicación interna.
Recomendaciones
Realizar monitoreo periódico de la red.
Mantener firewall y antivirus actualizados.
Aplicar políticas de acceso seguro.
Planificar futuras ampliaciones según las necesidades de la academia.
Conclusión

El proyecto muestra cómo una infraestructura informática bien organizada puede mejorar el funcionamiento de una academia de inglés, proporcionando conectividad, seguridad y acceso eficiente a los recursos compartidos.
